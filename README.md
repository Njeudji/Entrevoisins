# OpenClassrooms

Ce dépôt contient une mini-application pour le P3 du parcours **Grande École du Numérique**.

Build = Maj + F9
Run = Maj + F10
Copy = Ctrl + C
Paste = Ctrl + V
Cut = Ctrl + X
Search = Ctrl + F
Commit = Ctrl + X
Push = Ctrl + Maj + K
Update Project = Ctrl + T
New Class = Ctrl + N
New File Class = Ctrl + Maj + N
